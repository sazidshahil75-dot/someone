# 💖 Shahil's Love Website for Samia Marium Maisha 💖

## 🚀 Deploy on Railway

### Step 1 — Upload to GitHub
1. Create a free account at [github.com](https://github.com)
2. Create a **new repository** (name it anything, e.g. `love-website`)
3. Upload ALL these files into the repo:
   - `app.py`
   - `requirements.txt`
   - `Procfile`
   - `railway.json`
   - `.gitignore`
   - `templates/index.html`
   - `static/images/` ← put her photos here!

### Step 2 — Deploy on Railway
1. Go to [railway.app](https://railway.app) and sign up (free)
2. Click **"New Project"** → **"Deploy from GitHub repo"**
3. Select your repository
4. Railway will auto-detect Python and deploy it 🎉
5. Click **"Settings"** → **"Generate Domain"** to get your live URL

### That's it! 🌸
Your love website will be live at a URL like:
`https://love-website-production.up.railway.app`

## 📸 How to Add Her Photos
Place her photos inside `static/images/` before uploading to GitHub.

Supported formats: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`

## 🎵 Music
Place your music file inside `love_app/static/` and rename it to one of:
- `music.mp3` ✅ (recommended)
- `music.ogg`
- `music.wav`
- `music.m4a`

It will play automatically when she opens the website! 🎶
